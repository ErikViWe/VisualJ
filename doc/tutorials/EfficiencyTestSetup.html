Effizienzmessungen: Einführung

Nachdem Alice Herr Bobs Aufgaben gelöst hat, möchte sie die Performanz der Sortieralgorithmen vergleichen.

Hier sind nochmal die beiden Algorithmen: 

	public static void bubbleSort(int[] arr) {
		for (int i = 1; i < arr.length; ++i) {
			for (int j = 0; j < arr.length - i; ++j) {
				if (arr[j] > arr[j + 1]) {
					swap(arr, j, j + 1);
				}
			}
		}
	}
	
	public static void selectionSort(int[] arr) {
		for (int i = 0; i < arr.length - 1; ++i) {
			int min = i;
			
			for (int j = i + 1; j < arr.length; ++j) {
				if (arr[j] < arr[min]) {
					min = j;
				}
			}
			
			swap(arr, i, min);
		}
	}

Alles, was Alice tun muss, ist, den Zähler bei jedem Schleifendurchlauf um 1 nach oben zählen zu lassen (*1).

Die Anzahl der Schritte wird dann bei der Effizienzmessung ebenso wie die Laufzeit und der Speicherverbrauch gespeichert.
